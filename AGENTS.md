# Agent Instructions

Before editing any UI or frontend code:

1. Read `design.md`.
2. Use `/tokens/tokens.optimized.json` and `/styles/tokens.css`.
3. Use `/assets/icons/raw` for Catnip icons before introducing another icon set.
4. Reuse existing Catnip UI patterns first.

Never hardcode colors, spacing, radius, typography, icon sizes, icon colors, or shadows. All UI must follow the Catnip Design System.
