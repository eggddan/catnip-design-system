# Catnip Design System

Catnip design tokens, usage rules, and icon assets for building clean, light, playful, social, content-first UI.

## Files

- `design.md`: Codex-facing design system rules
- `AGENTS.md`: agent instructions for using the system
- `tokens/tokens.optimized.json`: optimized source tokens
- `styles/tokens.css`: CSS variables generated from tokens
- `styles/icons.css`: token-backed icon helper classes
- `assets/icons/raw`: Catnip SVG icon library
